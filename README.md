# World Cup Flag Bracket

국기 중심 월드컵 토너먼트 대진표 MVP입니다.

## 구조
- `index.html` — 메인 대진표
- `styles.css` — 반응형 UI
- `data.js` — 국가/대진/선수명단 샘플 데이터
- `teams/*.html` — 국가별 선수명단 페이지 placeholder

## 운영 방식
1. 공식 대진/선수명단 확정 후 `data.js`만 갱신합니다.
2. 광고 승인이 나면 `.ad-slot` 영역에 AdSense 코드를 삽입합니다.
3. FIFA/협회 공식 로고는 사용하지 않습니다.
